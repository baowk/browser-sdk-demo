# 科技风格设计指南

## 设计理念

本项目采用赛博朋克科技风格设计，营造未来感和专业感的用户界面体验。

## 核心设计元素

### 1. 配色方案
- **主色调**: 青色 (#00ffff) - 代表科技与未来
- **辅助色**: 紫色 (#8000ff) - 增加神秘感和深度
- **背景色**: 深色 (#0a0a0a) - 营造沉浸式体验
- **面板色**: 深蓝半透明 (rgba(15, 15, 25, 0.85)) - 现代感面板

### 2. 视觉特效

#### 发光效果
```css
/* 文字发光 */
text-shadow: 0 0 10px rgba(0, 255, 255, 0.7);

/* 边框发光 */
box-shadow: 0 0 20px rgba(0, 255, 255, 0.6);

/* 渐变发光 */
background: linear-gradient(90deg, transparent, rgba(0, 255, 255, 0.3), transparent);
```

#### 毛玻璃效果
```css
backdrop-filter: blur(20px);
background: rgba(15, 15, 25, 0.85);
```

#### 动画效果
- 按钮悬停辉光动画
- 浮动粒子动画
- 网格背景流动效果
- 文字呼吸灯效果

### 3. 字体规范
- **主要字体**: 'Courier New', monospace
- **字号规范**: 
  - 标题: 20-24px
  - 正文: 14px
  - 标签: 12px
  - 按钮: 12-14px
- **字母间距**: 1-2px (增加科技感)

### 4. 布局特点
- **容器宽度**: 420px (登录页面)
- **内边距**: 45px 35px
- **元素间距**: 统一28px
- **圆角**: 8-15px
- **边框**: 1px 半透明青色边框

### 5. 交互反馈
- **悬停效果**: 发光增强 + 微位移
- **点击效果**: 颜色加深 + 内阴影
- **焦点状态**: 青色高亮边框
- **禁用状态**: 降低透明度

## 组件设计规范

### 登录表单
- 全屏深色背景配网格动画
- 半透明面板配合毛玻璃效果
- 输入框带发光边框和悬停效果
- 按钮带扫描光动画效果

### 控制台界面
- 顶部状态栏显示用户信息和在线状态
- 导航标签带激活指示器和辉光效果
- 卡片组件采用悬浮设计和边框发光
- 进度条使用青紫渐变色彩

### 模态框
- 深色半透明遮罩层
- 发光边框和毛玻璃面板
- 统一的科技风格输入组件

## 响应式考虑
- 保持科技元素在不同屏幕尺寸下的一致性
- 小屏幕适当调整内边距和字体大小
- 确保关键信息的可读性

## 性能优化
- CSS动画使用GPU加速属性
- 合理使用backdrop-filter避免性能问题
- 动画复杂度适中，保证流畅体验

## 可访问性
- 确保足够的颜色对比度
- 保留键盘导航功能
- 文字大小适合阅读
